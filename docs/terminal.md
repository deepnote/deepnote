If you need to run some advanced scripts, installations or other tasks, you can use an integrated terminal directly in Deepnote. To create a new terminal, click the **+** button in the **Notebooks** section of the left sidebar and select **Create a new terminal** from the dropdown menu.

Beware, terminals are also collaborative so all collaborators will be able to view the commands (unless you close terminal with a right-click on it's name).

![terminal animation](https://media.graphassets.com/ePieNO0CRnSFAsPsvkm9)

<Callout status="info">
Terminals are an experimental feature so occasional bugs might happen.
</Callout>
