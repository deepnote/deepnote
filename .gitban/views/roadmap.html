      description
      A user can install one CLI and run, inspect, lint, validate, convert, diff, and analyze .deepnote (and .ipynb/.py/.qmd) files locally — with reactive execution that re-runs only the dependent blocks, venv auto-detection, and on-the-fly database integration credentials. Point pythonEnv/--python at that venv. Bun is only for the PyPI release build, not local dev.
